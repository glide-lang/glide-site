# Security Policy

This repository contains the static website (HTML/CSS/JS and assets) for glide-lang.org. It ships no server code.

## Reporting a vulnerability

- **In the site** (e.g. an XSS-prone snippet, a malicious link, or a compromised asset): please report it privately. Use GitHub's [private vulnerability reporting](https://github.com/glide-lang/glide-site/security/advisories/new) for this repo, or contact the maintainer [@Murilinho145SG](https://github.com/Murilinho145SG). Do not open a public issue for a security problem.

- **In the Glide language, compiler, or standard library**: report it against [glide-lang/Glide](https://github.com/glide-lang/Glide/security).

Please do not disclose a vulnerability publicly until it has been addressed. We aim to acknowledge reports within a few days.
